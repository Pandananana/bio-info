Background

In this assignment you will analyse spatial single-cell transcriptomic data generated with the CosMx Spatial Molecular Imager (NanoString/Bruker) from a mouse brain tissue section (coronal hemisphere).

Spatial transcriptomics enables the simultaneous measurement of gene expression and physical location of cells within a tissue section. Unlike bulk or dissociated single-cell RNA-seq, it preserves the spatial context of transcriptional heterogeneity — allowing us to ask not only which cell types are present, but where they reside within the tissue.

The CosMx platform preserves the spatial position of each cell within the tissue, enabling not only the identification of cell types but also the study of how they are physically organised within anatomical structures.

The dataset

The dataset uses a targeted CosMx Mouse Neuroscience RNA panel of ~950 genes measured across thousands of cells. Because the panel is already focused on biologically relevant targets, we deliberately do not perform Highly Variable Gene (HVG) selection before dimensionality reduction — filtering genes at this stage would risk discarding signal from rare cell populations.

CosMx data are generated using an imaging-based approach where RNA molecules are detected in situ via sequential fluorescent probe hybridisation cycles. Each cell retains its spatial \((x, y)\) position in the tissue throughout.

Dimensionality reduction

PCA

Because the CosMx panel is already targeted (~950 genes), we run PCA on all genes — no HVG selection is applied.

Why scale? Without scaling, highly expressed genes dominate the top PCs simply because they have larger absolute variance. ScaleData() centres each gene to mean 0 and scales to variance 1, ensuring every gene contributes equally to PCA.

# Make normalised counts available to Seurat's PCA (no re-normalisation)
obj[["RNA"]]$data <- obj[["RNA"]]$counts

# Scale all genes
GE <- ScaleData(obj, features = rownames(obj))
cat("Scaling done:", nrow(obj), "genes scaled.\n")

PC selection. PC standard deviations drop sharply over the first ~8 PCs, decline more gradually until ~PC 20, and then flatten onto a near-constant plateau (SD ≈ 1) that reflects technical noise rather than biological structure. We therefore retain the top 20 PCs for downstream UMAP and graph construction: this captures the clear biological signal while excluding the noise floor. Using fewer PCs (e.g. 10) risks merging closely related cell populations, while using many more (e.g. 50) injects noise into the neighbourhood graph without adding meaningful signal.

DimPlot(GE, reduction = "pca", dims = c(1, 2), pt.size = 1, alpha = 0.2) +

UMAP

UMAP is run on the top PCs (not on raw gene expression). It produces a 2D embedding that preserves local neighbourhood structure and is well-suited to visualising discrete cell populations.

Parameters:
dims: number of PCs to use — should match your elbow plot choice.
n.neighbors: size of the local neighbourhood (typical range: 15–50).
min.dist: controls how tightly cells are packed (lower = tighter clusters).

GE <- RunUMAP(GE,
              dims        = 1:50,   # adjust based on elbow plot
              n.neighbors = 30,
              min.dist    = 0.3,
              metric      = "euclidean",
              seed.use    = 42)

Deliverable

Cluster-level annotation

To interpret the unsupervised clusters biologically, two complementary annotations are computed:

  1. The top marker gene of each cluster (highest average log-fold-change from the FindAllMarkers Wilcoxon test above) is used as a transcriptional identity tag.
  2. The majority SingleR label (against the celldex Mouse RNA-seq reference) provides a broad cell-class call per cluster, but with the important caveat that this bulk-RNA reference contains only six classes and tends to over-call “Oligodendrocytes” for any white-matter-adjacent neuronal population — neuronal sub-types in particular cannot be reliably distinguished by this reference. We therefore use the cluster-level marker gene as the primary identity readout and rely on the colour-encoded cluster ID to expose the full transcriptional and spatial diversity of the dataset.
# Top-1 marker gene per cluster (largest avg_log2FC)
top_marker <- markers %>%
  group_by(cluster) %>%
  slice_max(order_by = avg_log2FC, n = 1, with_ties = FALSE) %>%
  ungroup() %>%
  select(cluster, marker = gene, avg_log2FC)

# Majority SingleR label per cluster (skipping NA)
majority_singler <- GE@meta.data %>%
  filter(!is.na(SingleR_label)) %>%
  count(seurat_clusters, SingleR_label) %>%
  group_by(seurat_clusters) %>%
  slice_max(n, n = 1, with_ties = FALSE) %>%
  ungroup() %>%
  transmute(cluster = seurat_clusters, singler = SingleR_label)

cluster_annot <- top_marker %>%
  left_join(majority_singler, by = "cluster") %>%
  mutate(label = paste0("C", cluster, " · ", marker))

Figure 1 | Unsupervised transcriptional clustering of CosMx mouse-brain cells recapitulates known anatomical organisation. Spatial transcriptomic data (CosMx Spatial Molecular Imager, ~950-gene Mouse Neuroscience panel) acquired on a single coronal hemisphere of mouse brain. After scaling and PCA on all panel genes (top 20 PCs retained), cells were embedded with UMAP and partitioned into 27 transcriptional clusters by Louvain community detection (k-NN = 30, resolution = 0.5) on the shared nearest-neighbour graph. Each cluster is shown in a distinct colour that is identical between panels, so the same cluster can be tracked from transcriptional space to physical space. (A) UMAP embedding of every cell. Results

Unsupervised clustering of 48,679 cells on the ~950-gene CosMx Mouse Neuroscience panel resolved 27 transcriptionally distinct populations that separate cleanly in UMAP space (Fig. 1A) and whose top marker genes span the canonical neuronal, glial, vascular and ependymal classes of the mouse brain — including layer-specific glutamatergic neurons (Slc17a7, Tle4), Sst and Vip interneurons, Drd1/Penk striatal medium-spiny neurons, the full Pdgfra → Bcas1 → Plp1 oligodendrocyte differentiation trajectory, three molecularly distinct astrocyte clusters and rare cell types such as Foxj1/Ttr-positive choroid-plexus epithelium. Projecting these clusters back onto the original tissue coordinates (Fig. 1B) reveals that the transcriptional diversity is laid out along the classical anatomy of the coronal mouse brain: glutamatergic-neuron clusters tile the cortical ribbon in clear laminar bands, Drd1/Penk clusters fill the striatum, Tcf7l2-positive cluster C3 is confined to the thalamus, mature Plp1-positive oligodendrocytes (C0) trace the corpus-callosum and internal-capsule white-matter tracts, and the small Foxj1/Ttr clusters outline the ventricular ependyma and choroid plexus. The tight match between unsupervised transcriptional structure and known mouse-brain cytoarchitecture demonstrates that a targeted, imaging-based panel of only ~950 genes is sufficient to jointly resolve molecular cell-type identity and the spatial cytoarchitecture in which those cells reside — the dual readout that motivates spatial transcriptomics over bulk or dissociated single-cell approaches when the goal is an anatomically-aware characterisation of the mammalian brain.
